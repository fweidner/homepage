My contribution is the mentioned fault-tolerance protocol which handles checkpointing as well as replaying events in case of the failure of one or more parts of the system.

We developed an Android application which displays objects in a so-called fish-tank environment. (Motive, C++, Android, Multi-Threading, WinSock)
